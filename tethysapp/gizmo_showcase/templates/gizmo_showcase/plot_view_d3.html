Plot View - D3 Renderer

Plot views can be used to generate plots of tabular data. Plot View has two rendering engines: Highcharts JS and D3.js. Both plotting libraries are powerful, but Highcharts is only free to use in certain circumstances.

For example code and an explanation of options see Gizmo Options Object API for Plot View.
