Table View

Table views can be used to display tabular data. The table view gizmo can be configured to have columns that are editable. When used in this capacity, embed the table view in a form with a submit button. Table View gizmos are styled with Bootstrap.

For example code and an explanation of options see Gizmo Options Object API for Table View.
